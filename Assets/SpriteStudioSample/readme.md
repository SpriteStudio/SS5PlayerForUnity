## サンプルプロジェクトについて  

SpriteStudioを使用したアクションゲーム風のサンプルプロジェクトです。  
Windowsでの動作を確認しています。  
含まれているファイルはライセンスに基づき使用してください。  

## ファイル構成  

- SpriteStudioSampleフォルダ  
mainscene.unityがメインのシーンファイルになります。  

- imageフォルダ  
背景の地形画像が含まれています。  

- Resourcesフォルダ  
ゲーム中に動的に生成するprefabが含まれています。  
ヒットエフェクトとしてeffect_comipo.sspjからコンバートしたprefabと、  
パーティクルシステムで作成したprefabが含まれています。  

- scriptフォルダ  
各種GameObjectを制御するためのスクリプトが含まれています。  

- soundフォルダ  
BGMをSEが含まれています。  

- SSdata  
  - BachImport.txt  
バッチインポート機能を使用で使用するテキストファイルです。  

  - SpriteStudioSSPJ  
SpriteStudio5で作成したアニメーションファイルが含まれています。  

  - character_template1
character_template1.sspjをインポートしたアニメーションprefabが含まれています。

## 操作方法  

- 移動  
キーボード矢印左右キー  

- ジャンプ  
キーボード矢印上キー  

- 攻撃  
キーボードZキー  

- 防御  
キーボードXキー
